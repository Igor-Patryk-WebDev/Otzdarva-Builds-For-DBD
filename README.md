# Otzdarva Builds For Dead by Daylight

A comprehensive library of **Dead by Daylight builds** created and recommended by Otzdarva.

The platform allows players to **browse curated killer and survivor builds**, discover different playstyles, and **explore perk combinations** based on Otzdarva's recommendations. All builds are organized using **up-to-date game data** and presented through a modern, user-friendly interface.

**Website Link:** https://otzdarva-builds.com/

## Version 2.0

Version 2.0 is a **complete rewrite** of our original project.

The application has been rebuilt from the ground up using a **modern web development stack** to a fully TypeScript-based architecture, featuring a redesigned user interface, **improved performance, enhanced maintainability, and a significantly better user experience**.

## Overview

Otzdarva Builds For DBD provides players with easy access to curated Dead by Daylight builds for both killers and survivors.

The platform organizes and presents build recommendations in a clear and searchable format, making it easy for players to discover new playstyles, learn perk synergies, and find builds tailored to specific characters.

All game-related information is automatically updated through a dedicated scraping service that retrieves data directly from the Dead by Daylight Wiki.

## Technology Stack

### Frontend

- React 19
- TypeScript
- TanStack Router
- TanStack Query
- TanStack Hotkeys
- Tailwind CSS

### Data Infrastructure

- Dedicated TypeScript-based web scraping service
- Automated data synchronization
- JSON-based game data pipeline
